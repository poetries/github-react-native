Github客户端
---

> 使用`React native`开发`GitHub`客户端，适配`IOS`、`Android`。

功能
---

- [x] 支持订阅多种编程语法
- [x] 支持添加/删除编程语言，并支持自定义它们的排序
- [x] 支持收藏喜欢的项目;
- [x] 支持多种颜色主题自由切换;
- [x] 支持搜索,并自持自定义订阅关键字;
- [x] 支持分享,轻松将自己喜欢的项目分享给好友;
